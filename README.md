Описание
Программа генерирует случайные цитаты из заранее заданного списка, отображает текущую цитату и сохраняет историю генераций. Есть возможность фильтровать историю по автору и теме.

Инструкция по запуску
1. Установите Python.
2. Скачайте репозиторий.
3. В терминале перейдите в папку проекта.
4. Запустите: `python main.py`.
5. Нажимайте "Сгенерировать цитату" для получения случайной цитаты.
6. Используйте фильтры для сортировки истории.

Примеры использования
- Генерация цитаты
- Фильтрация истории
