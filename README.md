Описание программы
Программа предназначена для учета прочитанных книг. Позволяет добавлять книги, фильтровать по жанрам и количеству страниц, сохранять и загружать данные.

Инструкция по запуску
1. Установите Python.
2. Склонируйте или скачайте репозиторий.
3. В командной строке перейдите в папку проекта.
4. Запустите: `python main.py`.

Использование
- Введите сведения о книге и нажмите "Добавить книгу".
- Для фильтрации выберите жанр и укажите минимальное число страниц, затем нажмите "Применить фильтр".
- Данные сохраняются автоматически при закрытии программы.
